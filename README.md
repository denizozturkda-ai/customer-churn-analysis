# Customer Churn Analysis

This project explores customer churn behavior using exploratory data analysis (EDA).
The goal is to identify key factors associated with customer attrition and provide insights that can help improve customer retention.

## Dataset
The dataset contains customer demographics, service usage, contract information, and billing details.

## Analysis Performed
- Data overview and cleaning
- Churn distribution analysis
- Churn by contract type, tenure, and monthly charges
- Identification of high-risk customer segments

## Key Findings
- Month-to-month contracts show significantly higher churn rates
- Customers with higher monthly charges are more likely to churn
- Short-tenure customers are at greater risk of leaving
- Electronic check payment users exhibit elevated churn risk
